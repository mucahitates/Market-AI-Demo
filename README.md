# SmartMarket AI – Finansal Piyasa Analiz Dashboardu

SmartMarket AI, gerçek piyasa verilerini kullanarak kripto para ve Borsa İstanbul (BIST) hisselerini analiz eden, görsel grafikler ve yapay zeka destekli yorumlar sunan bir finans dashboard uygulamasıdır.

Bu proje, Python ve modern veri görselleştirme araçları kullanılarak işlevsel, anlaşılır ve karar destek odaklı bir piyasa analiz sistemi geliştirme becerimi sergilemek amacıyla oluşturulmuştur.

---

##  Temel Özellikler

- Gerçek piyasa verisi entegrasyonu (Yahoo Finance)
- Kripto ve BIST hisse analizi
- Etkileşimli fiyat grafikleri (hover destekli)
- Yapay zeka destekli piyasa yorumu
- Yüzdelik değişim, trend ve volatilite analizi
- Hareketli ortalama hesaplaması
- TR formatında tarih seçimi
- Kullanıcı dostu ve sade arayüz

---

##  Yapay Zeka Analiz Mantığı

Sistem aşağıdaki göstergeleri analiz ederek yorum üretir:

- Fiyat değişim yüzdesi
- Kısa vadeli hareketli ortalama
- Volatilite seviyesi
- Momentum (fiyat hareket hızı)

Bu değerlere göre piyasa şu kategorilere ayrılır:

- Güçlü Yükseliş
- Belirgin Düşüş
- Yatay / Kararsız Piyasa

Amaç, kullanıcıya  “al/sat” sinyali vermek değil, piyasanın mevcut durumunu açıklayıcı ve anlaşılır şekilde yorumlamaktır.

-Projeyi çalıştırmak için:

streamlit run app.py

pip install streamlit yfinance altair matplotlib pandas
streamlit run app.py
